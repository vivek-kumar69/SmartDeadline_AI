# SmartDeadline AI

An AI-powered task management application that helps users prioritize tasks, assess deadline risks, and improve productivity using Google's Gemini AI.

---

## Features

- AI-based deadline risk analysis
- Smart task prioritization
- Interactive dashboard
- Task creation and management
- Productivity statistics
- Responsive React interface
- Express backend integration

---

## Tech Stack

### Frontend
- React 19
- TypeScript
- Vite
- CSS
- Lucide React

### Backend
- Node.js
- Express

### AI
- Google Gemini API

## Installation

Clone the repository

```bash
git clone https://github.com/vivek-kumar69/SmartDeadline_ai.git
```

Move into the project

```bash
cd SmartDeadline_ai
```

Install dependencies

```bash
npm install
```

---

## Environment Variables

Create a `.env` file in the project root.

```
GEMINI_API_KEY=YOUR_API_KEY
```

---

## Running Locally

Start the development server

```bash
npm run dev
```

Open your browser at

```
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Run production server

```bash
npm start
```
